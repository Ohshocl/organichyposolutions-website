Module 1 — Graphs, Functions & Linear Equations
1.1 GraphsCore
Coordinate Plane & Graphing Points

Every point is written as an ordered pair (x, y). The x-axis is horizontal, y-axis is vertical. The plane is divided into 4 quadrants: Q1 (+,+), Q2 (−,+), Q3 (−,−), Q4 (+,−).

Distance Formula
d = √[(x₂−x₁)² + (y₂−y₁)²]
Midpoint: M = ((x₁+x₂)/2, (y₁+y₂)/2)
Example
Distance from (1,2) to (4,6):
d = √[(4−1)² + (6−2)²] = √[9+16] = √25 = 5
1.2 FunctionsCore
What is a Function? Domain & Range

A function assigns exactly one output to each input. Test: each x-value appears only once in the table, or use the Vertical Line Test — if any vertical line crosses the graph more than once, it is NOT a function.

Domain = all allowed inputs (x-values). Range = all possible outputs (y-values).

Function notation: f(x) means "the function f evaluated at x." f(3) means plug 3 in for x. If f(x) = 2x + 1, then f(3) = 2(3) + 1 = 7.
Domain restrictions: You CANNOT divide by zero. You CANNOT take the square root of a negative number (in real numbers). Set denominator ≠ 0 or radicand ≥ 0 to find domain.
1.3 Linear FunctionsCore
Slope, Intercepts & Forms of a Line
Slope
m = (y₂ − y₁) / (x₂ − x₁) = rise / run

Three forms of a line:

Slope-Intercept: y = mx + b (m = slope, b = y-intercept) Point-Slope: y − y₁ = m(x − x₁) Standard: Ax + By = C
Parallel & Perpendicular
Parallel lines: SAME slope (m₁ = m₂)
Perpendicular: NEGATIVE RECIPROCAL slope (m₁ × m₂ = −1)
Horizontal line: y = k, slope = 0. Vertical line: x = k, slope = UNDEFINED (not zero!).
1.4 Linear Equations
Solving Linear Equations — 6-Step Process
  1. Clear fractions/decimals (multiply both sides by LCD)
  2. Use Distributive Property to remove parentheses
  3. Combine like terms on each side separately
  4. Move all variable terms to one side
  5. Move all constant terms to the other side
  6. Divide both sides by the coefficient of the variable
Special cases: If variables cancel and you get a TRUE statement (0 = 0) → infinitely many solutions. If you get a FALSE statement (3 = 5) → no solution.
Module 2 — Systems of Linear Equations
2.1 Systems of 2Core
Substitution & Elimination Methods

Substitution: Solve one equation for one variable, then substitute into the other.

  1. Isolate one variable in the simpler equation
  2. Substitute that expression into the other equation
  3. Solve for the remaining variable
  4. Back-substitute to find the other variable
  5. Check in BOTH original equations

Elimination (Addition): Multiply equations so that one variable's coefficients are opposites, then add the equations to eliminate that variable.

Solution types: One solution = intersecting lines. No solution = parallel lines (same slope, different y-int). Infinite solutions = same line.
2.2 Systems of 3
3×3 Systems — Gaussian Elimination

Use elimination repeatedly to reduce to a 2×2 system, then a 1×1:

  1. Pick any two pairs of equations, eliminate the SAME variable from each pair
  2. Now you have a 2×2 system — solve with substitution or elimination
  3. Back-substitute to find all three variables
  4. Check in all three original equations
2.3 Systems of Inequalities
Graphing Systems of Linear Inequalities

Graph each inequality separately. The solution is the overlapping (intersection) region.

  • Use a solid line for ≤ or ≥ (boundary is included)
  • Use a dashed line for < or > (boundary is NOT included)
  • Test the point (0,0) to determine which side to shade (unless it's on the line)
Module 3 — Polynomials & Quadratic Functions
3.1 Exponents & PolynomialsCore
Laws of Exponents & Polynomial Operations
Laws of Exponents
Product: xᵃ · xᵇ = xᵃ⁺ᵇ Quotient: xᵃ / xᵇ = xᵃ⁻ᵇ Power: (xᵃ)ᵇ = xᵃᵇ Zero: x⁰ = 1 (x ≠ 0) Negative: x⁻ⁿ = 1/xⁿ

FOIL for multiplying binomials: (a+b)(c+d) = ac + ad + bc + bd

Special Products
(a+b)² = a² + 2ab + b² (a−b)² = a² − 2ab + b² (a+b)(a−b) = a² − b²
3.3–3.7 FactoringCore
Factoring Decision Tree
  1. Always first: Factor out the GCF (Greatest Common Factor)
  2. 2 terms: Difference of squares: a² − b² = (a+b)(a−b)
    Sum of cubes: a³+b³ = (a+b)(a²−ab+b²)
    Diff of cubes: a³−b³ = (a−b)(a²+ab+b²)
  3. 3 terms (leading coeff = 1): Find two numbers that multiply to c and add to b in x²+bx+c
  4. 3 terms (leading coeff ≠ 1): Use the AC method — multiply a·c, find factors of ac that add to b, split the middle term, factor by grouping
  5. 4 terms: Factor by grouping — group in pairs
Sum of squares (a² + b²) does NOT factor over the reals! Only difference of squares factors.
3.3–3.6 Quadratic Graphs
Graphing Parabolas — Three Forms
Standard / General Form
f(x) = ax² + bx + c Vertex x-coordinate: x = −b/(2a)
Vertex Form
f(x) = a(x − h)² + k Vertex: (h, k) · Opens up if a > 0, down if a < 0
Intercept / Factored Form
f(x) = a(x − p)(x − q) x-intercepts (zeros): x = p and x = q
Module 4 — Rational Expressions & Equations
4.1–4.4 Rational ExpressionsCore
Simplifying, Multiplying, Dividing, Adding & Subtracting

Simplify: Factor numerator and denominator completely, then cancel common factors.

Multiply: Factor everything first, cancel, then multiply straight across.

Divide: Multiply by the RECIPROCAL of the second fraction (flip the second, then multiply).

Add/Subtract: Find the LCD, convert each fraction, then add/subtract numerators.

Domain restrictions: Always state what values make the denominator zero — those are excluded from the domain. State them from the ORIGINAL (before simplification).
4.5 Solving Rational Equations
Solving Rational Equations — Check for Extraneous Solutions
  1. Find the LCD of all fractions in the equation
  2. Multiply BOTH SIDES by the LCD to clear all fractions
  3. Solve the resulting polynomial equation
  4. CHECK each solution in the original equation — any value that makes a denominator zero is EXTRANEOUS and must be rejected
Extraneous solutions: Always check! When you multiply both sides by a variable expression, you may introduce false solutions.
Module 5 — Radical Expressions & Complex Numbers
5.1–5.6 RadicalsCore
Simplifying, Multiplying, Dividing & Rationalizing Radicals
Radical Rules
√(ab) = √a · √b √(a/b) = √a / √b ⁿ√(aᵐ) = a^(m/n)

To simplify √N: Find the largest perfect square factor of N, then take the square root of that factor outside.

Rationalizing: Multiply numerator and denominator by the radical (or conjugate for binomials) to eliminate radical from denominator.

Rationalize
3/√5 · (√5/√5) = 3√5/5
1/(2+√3) · (2−√3)/(2−√3) = (2−√3)/(4−3) = 2−√3
5.7 Radical Equations
Solving Radical Equations — Isolate & Square
  1. Isolate the radical on one side of the equation
  2. Square both sides (or raise to the nth power if nth root)
  3. Solve the resulting equation
  4. CHECK all solutions — squaring can introduce extraneous solutions!
Always check! If √(x) = −2, squaring gives x = 4, but checking: √4 = 2 ≠ −2. This is extraneous — no solution.
5.9 Complex Numbers
Imaginary & Complex Numbers
Imaginary Unit
i = √(−1), i² = −1, i³ = −i, i⁴ = 1 (repeating cycle)

A complex number has the form a + bi where a is the real part and b is the imaginary part.

Add/Subtract: Combine real parts and imaginary parts separately.

Multiply: Use FOIL, then replace i² with −1.

Divide: Multiply by the conjugate (a − bi) to get a real denominator.

Module 6 — Quadratic Equations
6.1 Solving QuadraticsCore
Four Methods for Solving Quadratic Equations

Method 1 — Factoring: Factor, set each factor = 0 (Zero Product Property)

Method 2 — Square Root Property: If x² = k, then x = ±√k. Use when no middle term.

Method 3 — Completing the Square:

  1. Move constant to right side
  2. Add (b/2)² to both sides
  3. Left side is a perfect square trinomial: (x + b/2)²
  4. Take square root of both sides (±)
  5. Solve for x

Method 4 — Quadratic Formula:

x = (−b ± √(b² − 4ac)) / (2a) for ax² + bx + c = 0
Discriminant b²−4ac: >0 → 2 real solutions · =0 → 1 real solution · <0 → 2 complex (imaginary) solutions
Module 7 — Exponential & Logarithmic Functions
7.3 Exponential FunctionsCore
Exponential Functions — Growth & Decay
Exponential Function
f(x) = bˣ where b > 0, b ≠ 1 Growth: b > 1 | Decay: 0 < b < 1

Key features: domain = all real numbers, range = (0, ∞), y-intercept = 1 (always!), horizontal asymptote y = 0.

Applications
Compound Interest: A = P(1 + r/n)^(nt) Continuous: A = Pe^(rt)
7.4–7.5 LogarithmsCore
Logarithms — Definition & Properties
Logarithm Definition
log_b(x) = y ↔ bʸ = x ln(x) = log_e(x) (natural log, base e ≈ 2.718) log(x) = log_10(x) (common log)
Log Properties
Product: log_b(MN) = log_b(M) + log_b(N) Quotient: log_b(M/N) = log_b(M) − log_b(N) Power: log_b(Mⁿ) = n · log_b(M) Change of Base: log_b(x) = log(x)/log(b) = ln(x)/ln(b)
To solve exponential equations: take log of both sides. To solve log equations: convert to exponential form and solve.
7.1–7.2 Composite & Inverse Functions
Function Composition & Inverse Functions
Composition
(f ∘ g)(x) = f(g(x)) — plug g(x) into f (g ∘ f)(x) = g(f(x)) — plug f(x) into g Note: f ∘ g ≠ g ∘ f in general!

Inverse function: Swap x and y, then solve for y. Notation: f⁻¹(x). The graph of f⁻¹ is the reflection of f over the line y = x.

A function has an inverse only if it is one-to-one — passes the Horizontal Line Test. f(f⁻¹(x)) = x and f⁻¹(f(x)) = x.
